## Quick Reference: Common Polyfills

| Function       | Key Concept                                      |
| -------------- | ------------------------------------------------ |
| `bind`         | Return function with preset `this` and args      |
| `call`         | Invoke with `this` and individual args           |
| `apply`        | Invoke with `this` and args array                |
| `map`          | Transform each element, return new array         |
| `filter`       | Keep elements passing test                       |
| `reduce`       | Accumulate to single value                       |
| `Promise.all`  | Resolve when all resolve, reject on first reject |
| `Promise.race` | Resolve/reject with first settled                |
| `debounce`     | Delay execution until pause in calls             |
| `throttle`     | Limit execution frequency                        |
| `deepClone`    | Recursively copy nested objects                  |
| `flat`         | Flatten nested arrays                            |

## Study Tips

1. **Practice coding without IDE** - Use whiteboard or paper
2. **Understand, don't memorize** - Know WHY code works
3. **Explain while coding** - Interviewers want to see thought process
4. **Test edge cases** - Empty arrays, null values, etc.
5. **Know time/space complexity** - Be ready to analyze Big O
6. **Build projects** - Real experience > theory

Good luck with your interviews!
